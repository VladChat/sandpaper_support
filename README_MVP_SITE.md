# eQualle Sandpaper Support MVP Site

Static GitHub Pages-ready MVP for the eQualle Sandpaper Support System.

## Included
- problem-first homepage
- problem search
- 10 major problem pages
- grit guide
- product support page
- how-to page
- documents placeholder
- AI assistant placeholder
- structured JSON data

## GitHub Pages base path
This MVP uses `/sandpaper_support/` because the repository is `VladChat/sandpaper_support`.
