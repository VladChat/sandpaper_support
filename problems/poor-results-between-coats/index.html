Poor Results Between Coats

Sanding between paint, primer, sealer, or clear coats leaves visible problems.

Which grit should I use between coats of paint?

The user needs a grit for light sanding between paint coats. Recommended grit: Use 320, 400, or 600 between coats, following the coating instructions.

What grit should I use between varnish coats?

The user needs a grit for light sanding between varnish coats. Recommended grit: Use 320, 400, or 600 depending on varnish instructions.

Dust nibs show in the finish

Small bumps or dust nibs appear in a painted or clear finish. Recommended grit: Use 400 or 600 lightly between coats where the coating instructions allow.
