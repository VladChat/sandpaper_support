How long should I sand with each grit?

The user does not know when to stop using one grit and move to the next. Recommended grit: Use each grit until the surface has an even scratch pattern, then move to the next finer grit.

Can I skip grits?

The user wants to know whether grit steps can be skipped. Recommended grit: Use gradual steps such as 80, 120, 180, 220, 320, 400, 600, and finer where needed.
