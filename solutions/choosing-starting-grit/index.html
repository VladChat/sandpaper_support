Problem

Choosing a starting grit

It is unclear which grit to start with.

Answer

Use 60, then 120, then 150, then 240, then 280, then 400, then 600 grit in order.

Why it happens
The task stage has not been identified yet.
Recommended grit
60-120 for removal, 150-240 for prep, 280-400 for finish prep, 600+ for fine stages.
Wet or dry
Dry for removal and prep; wet for fine finishing where suitable.
Success check
The grit removes the defect without creating unnecessary deep scratches.

What to do

  1. Identify whether you are removing, smoothing, finishing, or polishing prep.
  2. Start with the least aggressive grit that works.
  3. Move finer gradually.
  4. Inspect after each grit.
Avoid: Do not start with the coarsest grit unless heavy removal is needed.
