Problem

Using the assorted kit for heavy removal

The task needs fast material removal.

general sandpaper use grit selection 80 → 120 → 180 → 220 → 320 using the assorted kit for heavy removal the task needs fast material removal. general sandpaper use task needs fast material removal how to fix using the assorted kit for heavy removal not sure what grit to use
Answer

80 or 120, then refine with 180, 220, and 320 as needed. Start with 80 or 120 only if heavy removal is needed.

Why it happens
Starting too fine makes sanding slow.
Recommended grit
80 or 120, then refine with 180, 220, and 320 as needed.
Wet or dry
Dry.
Success check
Material is removed efficiently and then refined.

What to do

  1. Start with 80 or 120 only if heavy removal is needed.
  2. Stop once the defect is reduced.
  3. Refine the coarse scratches with intermediate grits.
  4. Finish at the grit needed for the next step.
Avoid: Do not leave 80-grit scratches as the final surface.
