Problem

Using the assorted kit for heavy removal

The task needs fast material removal.

Quick Answer

80 or 120, then refine with 180, 220, and 320 as needed. Start with 80 or 120 only if heavy removal is needed.

Best Grit Path

80 → 120 → 180 → 220 → 320

Step-by-Step Fix
  1. Start with 80 or 120 only if heavy removal is needed.
  2. Stop once the defect is reduced.
  3. Refine the coarse scratches with intermediate grits.
  4. Finish at the grit needed for the next step.
Why It Happens

Starting too fine makes sanding slow.

Wet or Dry

Dry.

Mistakes to Avoid
  • Do not leave 80-grit scratches as the final surface.
Success Check

Material is removed efficiently and then refined.
