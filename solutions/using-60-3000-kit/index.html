Problem

Using the 60-3000 assorted kit

The kit has many grits and the right order is unclear.

Answer

Use 80-120 for removal, 180-400 for prep, 600-3000 for fine finishing and polishing prep. Choose the grit range for the task stage.

Why it happens
Each grit range serves a different stage.
Recommended grit
Use 80-120 for removal, 180-400 for prep, 600-3000 for fine finishing and polishing prep.
Wet or dry
Dry for lower grits; wet for fine stages where suitable.
Success check
Each grit has a clear purpose in the sequence.

What to do

  1. Choose the grit range for the task stage.
  2. Do not use every grit if the surface does not need it.
  3. Avoid large jumps when scratches matter.
  4. Use high grits for fine refinement.
Avoid: Do not use 1000-3000 for heavy removal.
